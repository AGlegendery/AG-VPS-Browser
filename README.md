# AG VPS Browser  
یک مرورگر کامل داخل VPS برای دور زدن محدودیت‌ها

## آیا VPNهای عادی نیاز شما را پوشش نمی‌دهند؟

اگر فقط برای باز کردن چند سایت ساده VPN می‌خواهید، شاید VPNهای معمولی کافی باشند.  
اما خیلی وقت‌ها کارهایی وجود دارد که با VPN عادی درست انجام نمی‌شود.

مثلاً:

- دریافت **API‑ID و API‑Hash تلگرام**
- ثبت سرویس‌هایی که کشور را چک می‌کنند
- کار با سایت‌های **فوق تحریم** مثل Google Analytics
- ساخت اکانت در سرویس‌هایی که VPNهای عمومی را بلاک می‌کنند
- دانلود از سرویس‌هایی که IP دیتاسنتر خاص می‌خواهند
- تست پروژه یا ربات با IP خارج

در این حالت VPN معمولی معمولاً یکی از این مشکلات را دارد:

- IPها قبلاً بلاک شده‌اند  
- IP دیتاسنتر واقعی نیست  
- سرعت پایین است  
- ترافیک مشترک است  
- یا اصلاً اجازه دسترسی نمی‌دهد

اینجاست که **AG VPS Browser** وارد می‌شود.

# AG VPS Browser چیست؟

AG VPS Browser یک روش ساده است که به شما اجازه می‌دهد:

یک **مرورگر کامل لینوکس داخل VPS** اجرا کنید و از طریق مرورگر خودتان به آن وصل شوید.

در واقع شما:

- یک VPS موقت می‌سازید  
- اسکریپت پروژه را اجرا می‌کنید  
- یک **Linux Desktop + Browser کامل** داخل وب باز می‌شود  

و تمام ترافیک از **IP واقعی آن VPS** خارج می‌شود.

بنابراین:

- سایت مقصد IP واقعی دیتاسنتر را می‌بیند
- هیچ VPN عمومی در کار نیست
- بلاک شدن بسیار کمتر است

```Server

RAM: حداقل 2GB
```

نکته مهم:  
ما این سرور را **موقت** می‌گیریم و بعد از اتمام کار حذف می‌کنیم، بنابراین هزینه بسیار کم خواهد بود.

# قدم سوم — نصب AG VPS Browser

ابتدا git را نصب کنید:
```bash
apt update
apt install git -y
```
حالا پروژه را کلون کنید:
```
git clone https://github.com/AG-VPS-Browser/ag-vps-browser.git
cd ag-vps-browser
```
و اسکریپت نصب را اجرا کنید:
```
chmod +x install.sh
./install.sh
```
این اسکریپت به صورت خودکار:

Docker - را نصب می‌کند
- Webtop را اجرا می‌کند
یک Desktop لینوکس داخل مرورگر آماده می‌کند
قدم چهارم — # باز کردن مرورگر
بعد از چند دقیقه اسکریپت آدرس زیر را به شما می‌دهد:
```
http://SERVER-IP:3000
```
این آدرس را در مرورگر خود باز کنید.

حالا یک Linux Desktop کامل داخل مرورگر شما باز می‌شود.

داخل آن:

- Chromium
- Firefox
- Terminal
- File Manager

وجود دارد.

از اینجا می‌توانید:

- سایت‌های تحریم شده را باز کنید
- API بگیرید
- دانلود انجام دهید
- ثبت‌نام سرویس‌ها را انجام دهید

همه چیز با IP واقعی VPS انجام می‌شود.
